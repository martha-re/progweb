Tablas

Hay muchos sitios web  que muestran información como precios de acciones, resultados deportivos, datos de facturas y más. Estos datos son de naturaleza tabular, lo que significa que una tabla suele ser la mejor manera de presentar dichos datos.

Creación de Tablas

Antes de mostrar datos, primero se debe crear la tabla que contendrá los datos usando el elemento <table>.

<table>

</table>

Renglones

El primer paso para ingresar datos en la tabla es agregar renglones o filas utilizando el elemento de renglón de tabla: <tr>.

<tabla>
  <tr>
  </tr>
  <tr>
  </tr>
</table>
En el ejemplo anterior, se agregaron dos renglones a la tabla.

Datos

Los renglones no son suficientes para agregar datos a una tabla. También se debe definir cada elemento de celda utilizando el elemento de datos de tabla: <td>.

<tabla>
  <tr>
    <td> 20 </td>
    <td> 30 </td>
  </tr>
</table>

En el ejemplo anterior, se ingresaron dos datos (20 y 30) en el único renglón definido. Al agregar dos elementos de datos,  se crean dos celdas.

La salida sera:

20   30

Encabezados de tabla

Los datos de la tabla no tienen sentido sin títulos para describir lo que representan.

Para agregar títulos a renglones y columnas, se usa el elemento de encabezado: <th>.

El elemento de encabezado de tabla se utiliza como un elemento de datos de tabla. Al igual que los datos de una tabla, el encabezado debe colocarse dentro de una fila de la tabla.

<table>
  <tr>
    <th> </th>
    <th scope = "col"> Lunes </th>
    <th scope = "col"> Martes </th>
  </tr>
  <tr>
    <th scope = "row"> Temperaturas </th>
    <td> 73 </td>
    <td> 81 </td>
  </tr>
</table>
¿Qué pasa en el código anterior?

Primero, se agregó un renglón para contener los tres encabezados: un encabezado en blanco, un encabezado de Lunes y un encabezado de Martes. El encabezado en blanco crea la celda adicional de la tabla necesaria para alinear los encabezados de la tabla correctamente con los datos a los que corresponden.

En el segundo renglón, se agregó un encabezado de tabla como título del renglón: Temperaturas.

El uso del atributo scope, puede tomar uno de dos valores:

row: para indicar que el encabezado es para un renglón.
col: para indicar que el encabezado es para una columna.

Contorno de tablas

Hasta ahora, las tablas que se han creado han sido un poco difíciles de leer porque no tienen contorno.

En versiones anteriores de HTML, para agregar un borde o contorno a una tabla se utilizaba el atributo border con un valor entero, que representaba el grosor del borde.

<table border = "1">
  <tr>
    <td> 20 </td>
    <td> 30 </td>
  </tr>
</table>
El código del ejemplo anterior está obsoleto, su objetivo es ilustrar las convenciones más antiguas que se pueden encontrar al leer el código de otros desarrolladores.

Es probable que el navegador aún interprete este código correctamente, pero eso no significa que deba usarse.

Se utiliza CSS para agregar estilo a los documentos HTML, ya que ayuda a separar la estructura de una página de su apariencia. 

Se puede lograr el mismo efecto de borde de tabla usando CSS.

table, td {
border: 1px black solid;
}

Extension de columnas

¿Qué sucede si la tabla contiene datos que abarcan varias columnas?

Por ejemplo, un calendario podría tener eventos que abarquen varias horas o incluso varios días.

Los datos pueden abarcar columnas utilizando el atributo colspan. Los atributos aceptan un número entero (mayor o igual a 1) para indicar el número de columnas que abarca.

<table>
  <tr>
    <th> Lunes </th>
    <th> Martes </th>
    <th> Miércoles </th>
  </tr>
  <tr>
    <td colspan = "2"> Fuera de la ciudad </td>
    <td> De vuelta a la ciudad </td>
  </tr>
</table>
En el ejemplo anterior, Fuera de la ciudad abarca los encabezados de Lunes y Martes utilizando el valor 2 (dos columnas). De vuelta a la ciudad aparece solo bajo el encabezado Miércoles.

Extensión de renglones

Los datos también pueden abarcar varios renglones utilizando el atributo rowspan, con un valor entero (mayor o igual que 1) para indicar el número de renglones que abarcará .

<table>
  <tr> <! - Renglón 1 -->
    <th> </th>
    <th scope="col"> Lunes </th>
    <th scope="col"> Martes </th>
  </tr>
  <tr> <! - Renglón 2 -->
    <th scope="row"> Mañana </th>
    <td rowspan = "2"> Escuela </td>
    <td rowspan = "3"> Descanso </td>
  </tr>
  <tr> <! - Renglón 3 -->
    <th scope="row"> Tarde </th>
  </tr>
<tr> <! - Renglón  4 ->
  <th scope="row"> Noche </th>
  <td> Tarea </td>
</tr>
</table>

En el ejemplo anterior, hay cuatro renglones:

El primer renglón contiene una celda vacía y los dos encabezados de columna.
El segundo renglón contiene el encabezado del renglón Mañana, junto con Escuela, que abarca dos renglones debajo de la columna Lunes. La entrada "Descanso" que abarca tres renglones debajo de la columna del Martes.
El tercer renglón sólo contiene el encabezado del renglón Tarde.
El cuarto renglón contiene la entrada Noche,  la entrada Tarea ya que "Descanso" se extiende tres renglones.

Cuerpo de la tabla

Con el tiempo, una tabla puede crecer para contener una gran cantidad de datos y volverse muy larga. Cuando esto sucede, la tabla se puede seccionar para que sea más fácil de manejar.

Para seccionar una tabla se utiliza el elemento de cuerpo de tabla: <tbody>.

El elemento <tbody> debe contener todos los datos de la tabla, excluyendo los encabezados de la tabla.

Encabezado de tabla

Cuando se secciona el cuerpo de una tabla, también tiene sentido seccionar los encabezados de columna de la tabla con el elemento <thead>.

Pie de la tabla

La parte inferior de una tabla larga también se puede seccionar usando el elemento <tfoot>.

Ejemplo

Codigo HTML

<table border="1">
  <thead>
    <tr>
      <th> Trimestre </th>
      <th> Ingresos </th>
      <th> Costos </th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th> 1 </th>
      <td> $ 10 millones </td>
      <td> $ 7.5 millones </td>
    </tr>
    <tr>
      <th> 2 </th>
      <td> $ 12 millones </td>
      <td> $ 5 millones </td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <th> Total </th>
      <td> $ 22 millones </td>
      <td> $ 12.5 millones </td>
    </tr>
  </tfoot>
</table>
